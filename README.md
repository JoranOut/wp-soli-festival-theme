# Soli Festival Theme

~Plugin Name: wp-soli-festival-theme~
~Current Version:1.0.0~

A child theme of Twenty Twenty-Five for Soli's Binge Festival website. Clean, editorial design for presenting festival information, program details, and sponsors.

## Features

### Clean Editorial Design
Centered content layout with generous whitespace, system fonts, and a professional color palette inspired by modern newsletter design.

### Responsive Layout
Mobile-first responsive design with a narrow content column (653px) that works across all devices.

### Block Editor Ready
Built on Twenty Twenty-Five with customized `theme.json` settings for colors, typography, and layout that integrate with the WordPress block editor.

### GitHub Auto-Updates
Automatic updates via GitHub releases using the built-in theme updater.

## Installation

1. Ensure Twenty Twenty-Five is installed (parent theme)
2. Upload this theme to `/wp-content/themes/wp-soli-festival-theme/`
3. Activate the theme in WordPress admin

## Requirements

- WordPress 6.7+
- PHP 8.0+
- Twenty Twenty-Five parent theme
